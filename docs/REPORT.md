## 즉시 리포트 명령어

```
/report          전체 시장 즉시 리포트
/report kr       한국장만 즉시 리포트
/report us       미국장만 즉시 리포트
```

시간과 관계없이 명령 즉시 생성해 `#stock-analyze`에 발송한다.

## 리포트 필수 포함 항목

### 1. 오늘 시장 요약
- 전반적인 시장 분위기 (강세 / 약세 / 혼조)
- 주요 이슈 및 이벤트
- **시장 경제 뉴스** — 한국경제·매일경제·서울경제·머니투데이 RSS 헤드라인 상위
  4건(`core/market_data/news.py:fetch_market_news`). KR 리포트에만 실린다(US는
  종목별 Yahoo Finance 피드로 대체 — 시장 전반 피드 미보유).

### 2. 지수 현황

토스증권 API에 KOSPI/KOSDAQ/NASDAQ/S&P 500/DOW 지수 엔드포인트가 없어
"데이터 소스 미연동"으로 표기한다 (docs/TOSS_API.md 엔드포인트 표 기준).

### 3. 환율

| 통화쌍 | 현재 | 등락 |
|--------|------|------|
| USD/KRW | | |

토스증권 API는 USD/KRW 환율만 제공한다(`GET /api/v1/exchange-rate`, docs/TOSS_API.md) —
JPY 등 다른 통화쌍 엔드포인트가 없어 표기하지 않는다.

### 4. 공포탐욕지수 (Fear & Greed Index)
- 관심 종목 등락 비율(advance/decline breadth) 기반 0~100 대체 지표
  (`core/market_data/collector.py:_fear_greed_index`) — 토스 API에 시장 전체
  공포탐욕지수 엔드포인트가 없어 관심 종목 범위로 대체한다.

### 5. 토스 인기 종목 TOP 10
- 관심 종목 중 거래량 급증(`volume_ratio`) 상위 10종목
  (`core/market_data/collector.py:_popular_top10`) — 토스 API에 시장 전체 인기 종목
  랭킹 엔드포인트가 없어 관심 종목 범위로 대체한다.

### 6. 거래량 급증 종목
- 전일 대비 거래량 200% 이상 급증 종목

## 그래프 (matplotlib 자동 생성)

리포트 생성 시 아래 그래프를 자동 생성해 Discord에 파일로 첨부한다.
`core/report/chart.py`에서 생성하며 `logs/reports/charts/`에 저장한다.

| 그래프명 | 설명 |
|---------|------|
| 포트폴리오 수익률 | 날짜별 누적 수익률 라인차트 (LIVE/SIMULATION 각자의 스냅샷 이력, 2개 이상 쌓인 뒤부터) |
| 자산 변화 | 총 자산 KRW 추이 (위와 동일 조건) |
| 시장 지수 비교 | 관심 종목 일봉 종가의 동일가중 평균(KR vs US) 정규화 비교 — 토스 API에 KOSPI·NASDAQ 지수 엔드포인트가 없어 관심 종목 기반 대체 지표를 사용한다 |
| 보유 종목 비중 | 파이차트 |
| 업종 분포 | 섹터별 비중 바차트 — 토스 API에 종목 섹터 분류 데이터가 없어 **미구현** |
| 거래량 변화 | 관심 종목 거래량 히스토그램 |
| 수익 기여도 | 종목별 손익 기여 바차트 |
| 누적 수익률 | 시드 대비 누적 수익 라인차트 (위와 동일 조건) |

그래프 생성 실패 시 텍스트 리포트만 발송하고 `#stock-error`에 경고 Embed 전송.

## HTML 리포트 문서 (`core/report/html.py`)

정기·즉시·주간 리포트는 마크다운(Discord Embed용)과 **자기완결 HTML 문서**를 함께
생성한다. HTML은 대시보드 형태로, 위 그래프 PNG를 **base64 data URI로 인라인**하므로
외부 의존 없이 한 파일로 브라우저·디스코드 어디서든 그대로 열린다.

| 항목 | 내용 |
|------|------|
| 저장 위치 | `logs/reports/report_{market}_{날짜_시각}.html` (마크다운 `.md`와 같은 경로) |
| 구성 | KPI 요약(총자산·누적 수익률·현금 버퍼·보유 종목) → 시장별 현황(공포탐욕 미터·인기/급증 칩·RSI/신호/AI 표·**시장 경제 뉴스**) → 보유 종목 분석(**종목별 뉴스** 포함) → 그래프 → 주의 노트 |
| 주간 리포트 | 성과 지표 그리드 + 자금 정산 + 다음 주 방향 (`render_weekly_report`) |
| 테마 | 라이트/다크 자동 대응 (`prefers-color-scheme` + 뷰어 토글) |
| 전달 | `report_ready` 이벤트의 `htmlPath`로 실려 discord-bot이 `#stock-analyze`에 파일 첨부 (docs/INTERNAL_API.md) |
| 역할 분리 | 신호·추천·뉴스 판정은 `generator`, 표현(마크업·스타일)은 `html.py` — `html.py`는 `generator`를 import하지 않는다(순환 방지) |

마크다운·HTML·차트는 시장별 스냅샷을 **한 번만** 수집해 공유한다
(`generator._resolve_report_inputs`). HTML 생성 실패는 마크다운 발송을 막지 않는다.

## 일일·주간·월간 성과 지표

모든 성과 리포트에 아래 핵심 지표를 포함한다.

| 지표 | 설명 |
|------|------|
| 승률 (Win Rate) | 수익 거래 / 전체 거래 |
| 평균 수익률 | 거래당 평균 손익 |
| MDD (Max Drawdown) | 고점 대비 최대 낙폭 |
| 샤프 지수 (Sharpe Ratio) | 위험 대비 수익 |
| 수익 팩터 | 총 수익 / 총 손실 |
| 평균 보유 기간 | 종목당 평균 보유일 |
